# Exercício 1 — Função ou procedimento? (conceitual)
Para cada item abaixo, responda se é uma função (retorna um valor) ou um procedimento (só
executa uma ação) e justifique em uma frase:

    - Str(): Função. Retorna um valor numérico convertido em formato de texto (string).
    - QOut(): Procedimento. Não retorna nada, apenas executa a ação de imprimir algo no ecrã.
    - Date(): Função. Retorna o valor da data atual do sistema.
    - Len(): Função. Retorna o número exato (tamanho) de elementos de um array ou caracteres de uma string.
    - Upper(): Função. Retorna a string transformada em letras maiúsculas.
    - ClearScreen(): Procedimento. Apenas limpa visualmente o ecrã, sem devolver qualquer valor ao código.
