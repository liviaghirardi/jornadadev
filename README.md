# 🚀 Jornada Dev - Programa Start TOTVS

Bem-vindo(a) ao repositório da minha **Jornada Dev**! 

Este espaço foi criado para documentar toda a minha evolução, exercícios práticos e projetos desenvolvidos durante a minha participação no **Programa Start da TOTVS Paulista**.

## 📖 Sobre o Repositório
O objetivo deste repositório é registrar o meu aprendizado durante as semanas de imersão no programa. A trilha tem como foco principal o desenvolvimento de software voltado para o ecossistema corporativo da TOTVS.

## 🛠️ Tecnologias e Ferramentas
Durante o programa, estou desenvolvendo habilidades nas seguintes áreas:
- Lógica de Programação
- Harbour
- ADVPL (Advanced Protheus Language)
- Arquitetura do ERP Protheus
- Criação de Telas (CRUD) e Relatórios
- Programação Orientada a Objetos (OOP)

## 📚 Estrutura do Programa (Conteúdo)

Aqui você encontrará meus códigos divididos de acordo com as fases do treinamento:

- **[Fase 1]** Ambiente e Lógica de Programação (Configuração de ambiente, pseudocódigo e fluxogramas). - Finalizado!
- **[Fase 2]** Variáveis, Decisão e Repetição (Estruturas de repetição e tomada de decisão) - Finalizado!
- **[Fase 3]** Funções e Arrays (Blocos reutilizáveis e manipulação de listas).
- **[Fase 4]** Ambiente Protheus (Arquitetura do ERP, dicionário de dados e CRUD).
- **[Fase 5]** Menus, Relatórios e OOP (Validações e conceitos de orientação a objetos em ADVPL).
- **[Fase 6]** Projeto Final / TCC (Desenvolvimento de uma funcionalidade completa no Protheus, do dicionário ao relatório).
